# Pulido visual v4.0 - Producción

Ajustes realizados sobre el backoffice, sin modificar la lógica funcional ni el backend:

- Dashboard más compacto y ejecutivo.
- Tarjetas de indicadores con menor altura y mejor distribución.
- Bandeja de solicitudes con tabla más estable al 100% de zoom.
- Folio con tipografía monoespaciada para mejorar lectura.
- Acciones más limpias en la tabla de usuarios mediante iconos con tooltip.
- Mantenedor de portales y temáticas más compacto, evitando formularios excesivamente anchos.
- Ajustes de espaciado, cabeceras, filas y chips de estado.
- Build Angular validado correctamente con `npm run build`.

No se modificaron:

- Seguridad.
- Roles y permisos.
- Notificaciones por correo.
- Configuración pública/local.
- Backend.
- Migraciones Flyway.
