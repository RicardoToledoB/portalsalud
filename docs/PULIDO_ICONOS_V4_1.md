# Pulido visual v4.1 - Iconografía de acciones

Se ajustaron acciones administrativas para privilegiar iconos con tooltip sobre botones de texto extensos.

## Cambios

- Usuarios internos:
  - Guardar/crear usuario mediante botón compacto con icono.
  - Cancelar edición mediante icono.
  - Acciones de tabla mantienen iconos: editar, resetear clave, activar/desactivar.

- Portales y temáticas:
  - Guardar/crear portal mediante icono.
  - Editar portal mediante icono con tooltip.
  - Guardar/crear temática mediante icono.
  - Editar temática mediante icono con tooltip.

- Gestión de requerimiento:
  - Guardar gestión mediante icono.
  - Cancelar mediante icono.

## Validación

Frontend validado con:

```bash
npm run build -- --configuration development
```

Resultado: compilación Angular correcta.
